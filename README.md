# yuru-archive
![YuruArchive Logo](assets/yuruArchieve_Logo.png)

SpringBootを利用して、質問掲示板を作りました。


## 技術スタック
- Spring Boot 3.x
- Java 17
- PostgreSQL 16
- H2 Database(ロカルテスト用)
- Apache Tomcat 10.1
- Vue.js 3
- Docker / Docker Compose

## 特徴
- 質問・回答の投稿、編集、削除機能
- カテゴリ別の質問管理
- タグ機能
- 日本向け郵便番号APIによる住所自動入力
- 電話番号日本仕様のバリデーション対応
- Dockerを使用したローカル環境構築
