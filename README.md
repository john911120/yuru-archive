# yuru-archive
![YuruArchive Logo](assets/yuruArchieve_Logo.png)

SpringBootを利用して、質問掲示板を作りました。

## プロジェクト説明
🚀 本プロジェクト「ゆるアーカイブ」は、
週末のリラックスタイム中にふと浮かんだインスピレーションをもとに開発をスタートしました。
気軽に使える、柔らかい雰囲気の質問掲示板を目指しています。

## 技術スタック
- Spring Boot 3.x
- Java 17
- PostgreSQL 16
- H2 Database(ロカルテスト用)
- Apache Tomcat 10.1
- Thymeleaf
- Docker / Docker Compose

## 特徴
- 質問・回答の投稿、編集、削除機能
- カテゴリ別の質問管理
- タグ機能
- 日本向け郵便番号APIによる住所自動入力
- 電話番号日本仕様のバリデーション対応
- Dockerを使用したローカル環境構築
